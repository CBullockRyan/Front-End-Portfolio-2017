About this Page

Bootstrap

Bootstrap javascript and css files were used as frameworks to help the design process.

JQuery

JQuery was a library used to make the coding in javascript simpler.

JavaScript Source Files

Three JavaScript source files were used: data.js, helper.js, and pageBuilder.js. Data.js contained the data for the webpage, helper.js contained the html strings that would be appended, and pageBuilder contained the functions to build the webpage.

CSS Source Files

The only stylesheet used (besides bootstrap) was style.css which just contained all the data about how the page should look.

HTML Source Files

The only HTML file used was level1.html which contained all the structural information about the webpage.

Portfolio Files

The portfolio files contained all the html, css, and js files from every lab. This ws so that the URL to the different webpages created in those labs would work.
